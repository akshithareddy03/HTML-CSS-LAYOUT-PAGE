# HTML-CSS-LAYOUT-PAGE

## Project Overview

This project is a responsive multi-section static webpage developed using HTML5 and CSS3. It demonstrates core front-end concepts including semantic HTML, Flexbox, CSS Grid, responsive design, positioning, and the CSS box model.

## Features

- Responsive Navigation Bar
- Hero Section
- About Section
- Services Section
- Statistics Section
- Contact Form
- Footer

## Concepts Implemented

- Semantic HTML5
- CSS Flexbox
- CSS Grid
- Relative Positioning
- Absolute Positioning
- Margin & Padding
- CSS Box Model
- Responsive Design

## Technologies Used

- HTML5
- CSS3
